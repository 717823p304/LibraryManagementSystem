LIBRARY MANAGEMENT SYSTEM

A web-based Library Management System developed using Java, Servlets, JDBC, and MVC architecture.
This application allows administrators to manage books and authors efficiently using a structured layered approach (Bean → DAO → Service → Servlet).

PROJECT OVERVIEW

The Library Management System is designed to:
1. Add new books to the system
2. Manage author details
3. View book records
4. Validate and handle user input
5. Interact with a relational database using JDBC
The application follows a clean architecture structure to maintain separation of concerns and scalability.
